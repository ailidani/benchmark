Distributed Benchmark

Quick overview of DB

Distributed Benchmark is a micro-benchmark framework similar to YCSB, with more features and fine control over a cluster of clients running on multiple machines. DB code base is kept as simple and compact as possible, so that anyone can easily extend the system.
